## Live demo: [kolumb.tk/asteroids](https://kolumb.tk/asteroids/)

### Controls:
| Key | Action |
| --- | --- |
| `W` / `↑` | Thrust |
| `A` `D` / `←` `→` | Rotate |
| `Space` | Shoot |
| `E` | Autoshoot |
| `F` | Laser |
| `P` | Pause |
| `I` | Debug info |
| `R` | Restart on GameOver |
| `?` | Show controls |
